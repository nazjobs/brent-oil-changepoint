# Contributing to Birhan Energies Dashboard

We welcome contributions! Please follow these steps:

1.  **Fork** the repository.
2.  **Clone** your fork locally.
3.  **Install Dependencies**: `make setup`
4.  **Run Tests**: `make test`
5.  **Create a Branch**: `git checkout -b feature/amazing-feature`
6.  **Commit & Push**: Submit a Pull Request.

## Code Style
- Use Python docstrings.
- Ensure all new features have accompanying tests.
